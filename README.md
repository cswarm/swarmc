# swarmc

A lighweight computercraft emulator using Lua 5.2.4


## Features

 * Emultates computercraft
 * Hardware accelerated crypto using OpenSSL
 * Full control over Lua VM
 * Share Lua Stack over redis.

## Usage

```bash
  npm Install
  node index.js

  # this runs CraftOS shell for now.
```
